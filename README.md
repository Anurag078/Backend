# Backend Developer Assignment

## Features
- JWT Authentication
- Role-based access (Admin/User)
- CRUD APIs for tasks
- Protected routes

## Tech Stack
- Node.js
- Express
- MongoDB
- React

## Setup Instructions

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev

## API Endpoints

POST /api/v1/auth/register  
POST /api/v1/auth/login  
GET /api/v1/tasks  

## Scalability
- Can be extended to microservices
- Redis caching can improve performance
- Load balancing for high traffic
